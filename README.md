# 사용법

1. 크롬 웹 스토어에서 [Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)를 설치한다.
2. [src.js](https://github.com/PseudoCardiac/Tampermonkey-ChatGPT-PFP/blob/main/src.js)를 다운로드한다.
3. Tampermonkey 대시보드에 `src.txt` 파일을 드래그 앤 드롭하여 설치한다.
4. ChatGPT PFP 스크립트를 클릭헤 에디터를 연다.
5. 10번째 줄의 `PASTE IMAGE URL HERE` 텍스트를 원하는 이미지의 URL로 대체한다.
6. 저장한다.
